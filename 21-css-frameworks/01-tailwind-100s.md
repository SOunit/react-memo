# points

- bootstrap took high-level-component approach, offering `btn`, `dropdown`, `form`
- tailwind took functional approach
  - props
    - `flex`
    - `hover`
  - conditional styling
    - `lg`
    - `md`
    - `sm`
  - handle duplicate code
    - use js framework
    - use @apply
  - performance
    - big on development
    - small on production
  - VSCode tool
    - auto completion
